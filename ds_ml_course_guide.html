// 01 — The Roadmap

A step-by-step path from beginner to job-ready data scientist

01
Python & Programming Foundations 3–4 WEEKS

Build a rock-solid programming base. No prior coding experience needed. We go from "Hello World" to writing reusable, clean functions that power real data pipelines.

Python syntax & data types Lists, dicts, loops Functions & OOP basics File I/O NumPy & Pandas Bash & Linux CLI Jupyter Notebooks
02
Mathematics for Machine Learning 4–5 WEEKS

You don't need to be a mathematician, but you do need intuition. We build the essential math toolkit — the same foundations I used in my PhD and applied daily in industry.

Linear Algebra Matrix operations Calculus & gradients Probability theory Statistics & distributions Hypothesis testing Bayes' theorem
03
Data Analysis & Visualisation 3 WEEKS

Data rarely arrives clean. Learn to explore, clean, and tell stories with data — the skill that separates good analysts from great data scientists.

Exploratory Data Analysis Feature Engineering Handling missing data Matplotlib & Seaborn Correlation & causation SQL for data analysis t-SNE & PCA for EDA
04
Classical Machine Learning 5–6 WEEKS

The workhorse algorithms every data scientist must master. We go deep on both supervised and unsupervised methods — understanding not just how to run them, but why they work.

Linear & Logistic Regression Decision Trees Random Forest SVM LDA k-Means Clustering PCA Ridge & Lasso Model evaluation & cross-validation Scikit-learn
05
Deep Learning & Neural Networks 5–6 WEEKS

From perceptrons to convolutional networks. Understand backpropagation from first principles, then build real models using TensorFlow and PyTorch.

Feedforward NNs Backpropagation CNNs (Image Processing) RNNs & LSTMs Batch normalization Dropout & regularization TensorFlow & PyTorch Keras Hyperparameter tuning
06
NLP, Transformers & Generative AI 4–5 WEEKS

The most in-demand skill in AI right now. Learn how modern language models work, and build applications using LLMs, RAG, and Agentic AI frameworks — the exact stack I use at Broadridge.

Text preprocessing & tokenization Word embeddings Attention mechanism Transformer architecture BERT & fine-tuning LLMs & prompting RAG systems Agentic AI Vector databases (pgvector)
07
MLOps & Cloud Deployment 3–4 WEEKS

Building a model is only half the job. Learn to deploy, monitor, and scale ML systems in production — skills that make you immediately valuable on any ML team.

AWS SageMaker Azure ML MLflow experiment tracking Docker & containerization Model monitoring CI/CD for ML APIs with FastAPI S3 & cloud storage
08
Capstone Projects & Career Preparation 4 WEEKS

Apply everything you've learned to build 3 end-to-end portfolio projects — in finance, healthcare, and NLP — plus a complete job-hunt toolkit tailored to the DS/ML market.

Finance: Forecasting model Genomics: Classification pipeline NLP: Document extraction agent GitHub portfolio DS interview prep Resume & LinkedIn Case study walkthroughs
